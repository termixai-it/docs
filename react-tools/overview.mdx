# @termix-it/react-tool

A powerful, configurable AI chat interface component library designed for React applications. This library provides everything you need to integrate advanced AI conversations with function calling capabilities, knowledge base integration, and smart contract execution.

## Key Capabilities

### AI-Powered Conversations
- **Streaming responses** for real-time chat experience
- **Context-aware** conversations with message history
- **Customizable personalities** and model configurations
- **Usage tracking** with token and cost information

### Advanced Function Calling
- **REST API calls** with automatic execution and response formatting
- **Smart contract interactions** with MetaMask integration
- **Knowledge base search** for contextual information retrieval
- **Custom function definitions** for specialized use cases

### Smart Contract Integration
- **Automatic detection** of contract function calls
- **Multi-chain support** for Ethereum and EVM-compatible networks
- **Transaction handling** with confirmation tracking
- **Wallet connectivity** through MetaMask and web3 providers

### Developer Experience
- **TypeScript first** with comprehensive type definitions
- **Flexible theming** using CSS custom properties
- **Event callbacks** for monitoring and customization
- **Responsive design** that works on all devices

## Quick Preview

Here's what a basic implementation looks like:

```tsx
import { ChatInterface } from '@termix-it/react-tool';
import '@termix-it/react-tool/dist/styles.css';

function App() {
  return (
    <div className="h-screen">
      <ChatInterface
        projectId="your-project-id"
        aiConfigId="your-ai-config-id"
        apiBaseUrl="/api/proxy"
        authorization="Bearer your-jwt-token"
        welcomeMessage="Hello! How can I help you today?"
        enableKnowledgeBase={true}
        enableFunctionCalls={true}
        enableReActPattern={true}
        onFunctionExecuted={(call, result) => {
          console.log('Function executed:', call.name, result);
        }}
      />
    </div>
  );
}
```
