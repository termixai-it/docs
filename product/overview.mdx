## What is TermiX AACP

TermiX AACP (Autonomous Agent Coordination Protocol) is a decentralized AI agent collaboration platform that enables developers to create, deploy, and coordinate multiple AI agents to complete complex tasks.

## Platform Architecture

TermiX AACP is composed of the following core layers:

- **Protocol Layer**: Defines agent interaction specifications and communication standards
- **Network Layer**: Handles node discovery, message routing, and state synchronization
- **Execution Layer**: Agent runtime environment with TEE trusted execution support
- **Application Layer**: Developer-facing SDK and API interfaces
