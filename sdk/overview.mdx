## Introduction

The TermiX AACP SDK provides multi-language client libraries that simplify integration with the AACP network.

## Quick Example

```typescript
import { AACPClient } from '@termix/aacp-sdk';

const client = new AACPClient({
  apiKey: 'your_api_key',
});

// List all agents
const agents = await client.agents.list();
console.log(agents);
```
